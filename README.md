# Vision Impact Object Detection Application

## Overview
The Vision Impact Object Detection Application is designed to recognize and identify objects in images and videos in real time. Utilizing state-of-the-art machine learning algorithms, this application aims to assist users in various fields such as security, retail, and autonomous vehicles.

## Features
- **Real-time Object Detection**: Detects multiple objects with high accuracy in real time.
- **User-friendly Interface**: Simple and intuitive UI for seamless interaction.
- **Cross-platform Support**: Works on various platforms including Windows, macOS, and Linux.

## Technologies Used
- **Programming Language**: Python
- **Libraries**: TensorFlow, OpenCV, NumPy
- **Frameworks**: Flask for backend services

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aayushapandey02-ctrl/Student.git
   cd Student
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute:
```bash
python app.py
```
